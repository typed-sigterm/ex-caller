简约风格的随机点名工具。

## 主要功能

![](./static/demo.webp)

- 🔄️ 随机点名，可自定义过号时间
- 📓 多个名单，自由切换
- 🔧 名单批量导入/导出
- 🎉 彩带特效
- 🧩 自定义背景图片
- 🤫 点名计划 [^1]
